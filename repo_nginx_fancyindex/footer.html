Informations

Les informations concernant la façon dont ont été construites les images Debian/YunoHost sont disponibles sur build.labriqueinter.net. Vous pouvez reconstruire les images vous-mêmes si vous le souhaitez.

Guide express

Il suffit de télécharger le SD Card Installer, et l'exécuter (ajouter -2 à la fin pour une LIME2, ou -h pour voir les autres options) :

% bash install-sd.sh

Pour l'accès SSH et la post-installation de YunoHost, voir la section Post-installation ci-dessous.

Guide express (avec chiffrement)

La création d'une Brique utilisant un système de fichier chiffré n'est plus supportée pour le moment. Contacter l'équipe de développement si vous souhaitez aider sur ce sujet.

Post-installation

Pour finaliser la configuration de la Brique, il faut la faire démarrer sur la carte SD puis s'y connecter. Vous pouvez utiliser le script d'installation pour trouver la Brique sur votre réseau local :

% bash install-sd.sh -l

Une fois la Brique localisée, connectez-vous par SSH sur le port 22 :

root:yunohost

Note : S'il n'y a qu'une seule Brique sur votre réseau, vous pouvez aussi directement utiliser ssh root@fe80::42:acab%eth0 (en remplaçant eth0 par l'interface réseau que vous utilisez sur votre ordinateur).

La première chose à faire est de mettre à jour Debian/YunoHost :

% sudo apt-get update && sudo apt-get dist-upgrade

Vous pouvez maintenant faire la post-installation.

Ajoutez ensuite la liste des apps community à YunoHost :

% yunohost app fetchlist -n community -u https://app.yunohost.org/community.json

À partir de l'administration web de YunoHost, vous pouvez désormais installer les apps VPN Client et Wifi Hotspot, puis aller les configurer.

Hypercube

La méthode d'installation recommandée consiste à utiliser un fichier hypercube généré via cette page.

Le fichier hypercube peut être fourni au script install-sd.sh de la manière suivante :

% bash install-sd.sh -y fichier_hypercube.hyp

Si le fichier n'a pas été fourni via le install-sh.sh, il est plus tard possible de brancher une clef USB (formattée en NTFS ou FAT32) sur la Brique, contenant le fichier. L'installation via l'hypercube sera alors lancée automatiquement.

Le déroulement de l'installation de la Brique peut être suivi via l'interface web accessible à https://192.168.x.y:2468/

Ou bien il est possible de détecter automatiquement l'interface de debug sur le réseau à l'aide de cette page.
